About This Application

This application is designed to assist researchers in the IITA Yam Breeding Unit in extracting common biomarkers from multiple Hapmap files using accession numbers and converting files between different bioinformatics formats.

Key Features

  • ✅ Check if accession numbers are present in the database
  • ✅ Identify common biomarkers across multiple files
  • ✅ Download results in various formats: Hapmap, VCF, and Dosage
  • ✅ Convert files between Hapmap, VCF, and Dosage formats

How It Works

This app allows you to extract common biomarkers from multiple hmp files using accession numbers.

Follow these simple steps:
  1. Enter the accession numbers you want to extract (space-separated for multiple)
  2. Click "Check Files" to verify accession numbers in the database
  3. Review found files and check for common biomarkers
  4. Download results in your preferred format (Hapmap, VCF, or Dosage)
  5. Choose to download all in one file or separate files

File Format Converter

Convert Hapmap files to VCF or Dosage format with ease

📋 Conversion Types

Choose the type of file conversion you need from the options below:

  • 🔄 Hapmap haploid → Hapmap diploid - Convert haploid Hapmap files to diploid format
  • 🔄 Hapmap → VCF - Convert Hapmap format to Variant Call Format (VCF)
  • 🔄 VCF → Dosage - Convert VCF files to dosage format
  • 🔄 Hapmap → Dosage - Convert Hapmap files directly to dosage format
  • 🔄 VCF → Hapmap - Convert VCF files back to Hapmap format

ℹ️ Format Requirements

  • Hapmap haploid → Hapmap diploid: Requires .hmp.txt file
  • Hapmap → VCF: Requires .hmp.txt file
  • VCF → Dosage: Requires .vcf file
  • Hapmap → Dosage: Requires .hmp.txt file
  • VCF → Hapmap: Requires .vcf file
